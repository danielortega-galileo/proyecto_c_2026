# Proyecto C


## Descripcion
Este es un programa de c que funciona como conversor de monedas

## Prerequisitos 
1.Instalar C en ubuntu
  sudo apt install build-essential
2.Comprobar la version de C
  gcc --version


## Pasos para su ejecucion

1.Crear la carpeta para el proyecto

2.Escribir el codigo de C en la carpeta

3.Compilar el archivo para que genere el ejecutable

4.Usar el archivo ejecutable
